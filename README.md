## 🛠 Stack Technique

L'application est une Single Page Application (SPA) construite avec des outils modernes et typés :

* **Framework :** React 18
* **Build Tool :** Vite.js (pour une compilation ultra-rapide)
* **Langage :** TypeScript (pour la sécurité du typage)
* **Styling :** Tailwind CSS v4 (approche utility-first)
* **Icônes :** React Icons

## ⚙️ Architecture et Déploiement

Le site est hébergé de manière autonome sur un serveur privé virtuel (VPS) Hostinger sous Ubuntu. L'infrastructure de production démontre une maîtrise des environnements Linux et des outils DevOps de base :

* **Serveur web / Reverse Proxy :** Caddy Server (conteneurisé via Docker)
* **Gestion SSL :** Certificats HTTPS générés et renouvelés automatiquement via Let's Encrypt (intégré à Caddy)
* **Process de build :** Génération statique (`npm run build`) pour un rendu performant, sans charge serveur.
